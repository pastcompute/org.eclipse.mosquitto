Eecloud
=========

Eecloud is an open source implementation of a server for version 3.1 and
3.1.1 of the MQTT protocol.

See the following links for more information on MQTT:

http://mqtt.org/
http://www.ibm.com/developerworks/webservices/library/ws-mqtt/index.html

Eecloud project information is available at the following locations:

http://eecloud.org/ (main homepage)
https://bugs.eclipse.org/bugs/buglist.cgi?product=Eecloud (Find bugs)
https://bugs.eclipse.org/bugs/enter_bug.cgi?product=Eecloud (Submit bugs)
http://git.eclipse.org/c/eecloud/org.eclipse.eecloud.git/ (source code repository, web interface

There is also a public test server available at http://test.eecloud.org/

Eecloud was written by Roger Light <roger@atchoo.org>
